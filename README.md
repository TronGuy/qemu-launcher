
## Dependências

O projeto usa o tk/tkinter. Verifique se o pacote Tk está instalado.

> **Ubuntu/Derivados**:
> * sudo apt-get install python-tk ou sudo apt-get install python-tk
> 
> **Arch Linux**:
> * pacman -Sy tk ou sudo pacman -Sy tk
